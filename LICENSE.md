## 🛠️ Troubleshooting Common Issues

**❌ Cosmic Executor does not attach to Roblox after launching**
This usually happens when Roblox updates its client and the injection offset table becomes outdated. The fix is straightforward.
**Solution:** Check the **Cosmic Executor Website** for a hotfix update. If none is available yet, increase the `inject_delay_ms` value to `8000` in your `config.json` and restart both applications.

**❌ Windows Defender quarantines the Cosmic Executor installer**
Antivirus software frequently flags injection-based tools as potentially unwanted programs, even when the files are clean and verified.
**Solution:** Add `C:\Program Files\CosmicExecutor` to your Windows Defender exclusion list via `Settings → Virus & Threat Protection → Manage Settings → Exclusions`. Always verify the SHA-256 checksum from the **Cosmic Executor Website** before running any installer.

**❌ Scripts fail with "Execution Timeout" error**
Some complex scripts exceed the default 120-second execution window that Cosmic Executor enforces to prevent infinite loops.
**Solution:** Open your `config.json` and increase `script_timeout_seconds` to `300` or higher. If the script still fails, it may contain an unintended infinite loop.

**❌ Android overlay does not appear over Roblox**
On certain Android devices, the system aggressively restricts overlay permissions for sideloaded applications like Cosmic Executor.
**Solution:** Navigate to `Settings → Apps → Cosmic Executor → Appear on Top` and ensure the toggle is enabled. Also disable battery optimization for the app.

**❌ Cloud sync conflicts when editing scripts on multiple devices**
Prior to version 4.2.1, Cosmic Executor could overwrite changes if the same script was modified on two devices before syncing.
**Solution:** Update to the latest version of Cosmic Executor, which includes automatic conflict detection and merge prompts.
